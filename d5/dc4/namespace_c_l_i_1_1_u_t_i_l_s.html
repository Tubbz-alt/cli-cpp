std::string CLI::UTILS::Format_Command_String ( const std::string &  input_data)

Format a string into proper command format.

This method will take an input string and format it to look like what the command alias input wants. In essence, it will split the string into components with spaces being the delimiter. Then it will re-merge them such that there is one space between them.

The following example should illustrate this.

  • Input: " ls -lf " Output: "ls -ls"
Parameters
[in]input_dataString input to format.
Returns
Formatted String.
