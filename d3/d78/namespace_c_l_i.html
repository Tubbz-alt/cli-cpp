enum CLI::CLI_Event_Type : int32_t
strong

Enumerates different events that may be registered or triggered within CLI-CPP.

Enumerator
CLI_VARIABLE_SHOW 

Event for Print the Variable List.

CLI_VARIABLE_ADD 

Event for Adding a Variable to the Command-Parser.

CLI_VARIABLE_RM 

Event for Removing a Variable from the Command-Parser.

CLI_NULL 

Treat a null or no-op value.

CLI_PAUSE 

User requested to pause the CLI.

CLI_SLEEP 

User requested to sleep for a period of time.

CLI_REFRESH 

Event for refreshing the interface.

CLI_ALIAS_LIST 

Event for showing the alias list screen.

CLI_LOG 

Event for showing the log output screen.

CLI_HELP 

Event for showing the help windows.

CLI_BACK 

Event for returning to the main window.

CLI_SHUTDOWN 

Event for disconnecting the CLI.

Unknown event. Treat this as an error.
